# BurnCloud Client API - 项目概述

## 项目简介

BurnCloud Client API 是一个基于 Rust 和 Dioxus 框架开发的桌面应用程序，专门用于管理和监控大模型 API 调用平台。该应用提供了直观的用户界面来监控 API 端点状态、配置接口参数，并管理与各种大语言模型的集成。

## 核心特性

### 🚀 主要功能
- **API 端点监控**: 实时查看 API 服务状态
- **接口管理**: 统一管理各类大模型 API 接口
- **状态可视化**: 清晰的状态指示器显示服务运行情况
- **桌面应用**: 原生桌面体验，响应迅速

### 🎯 支持的 API 端点
- `/v1/chat/completions` - 对话完成接口
- `/v1/models` - 模型列表接口

## 技术栈

### 前端框架
- **Dioxus 0.6.3**: 现代化的 Rust UI 框架
- **自定义 CSS**: 完整的设计系统和样式库

### 开发语言
- **Rust 2021 Edition**: 高性能、内存安全的系统编程语言

### 应用类型
- **桌面应用**: 使用 Dioxus Desktop 构建的原生桌面应用

## 应用界面

### 主窗口配置
- **标题**: "BurnCloud API管理"
- **尺寸**: 1200×800 像素
- **响应式布局**: 适配不同屏幕尺寸

### UI 组件
- **页面头部**: 显示应用标题和描述
- **API 卡片**: 展示各个端点的状态信息
- **状态指示器**: 实时显示服务运行状态
