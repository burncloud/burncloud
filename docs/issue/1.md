### F. system.md 非 CSS 项（无自动 gate）

全 CSS 时不能忽略：

- [ ] **R1 i18n**：控制台硬编码英文（如 connect 里 `"45ms"`、`"842 Nodes"`）
- [ ] **四态**：空状态无引导、错误无「发生什么 + 怎么办」（J3 P0）
- [ ] **五态**：focus 键盘可见、disabled 样式
- [ ] **响应式**：Desktop / Tablet / Mobile 无 CI
- [ ] **动态色**：`--bc-dynamic-*` 的 `style:` 允许；勿用 inline 写间距/字号/裸 hex

## 建议修复顺序

1. **Phase 0** — 稳定 E2E（session、loop.lock、focus metrics），跑通 home-only `jobs-aesthetic`
2. **Phase 1** — 扩展 L1：`bc-font-*` / `bc-grid-*`；热点 connect → monitor → users
3. **Phase 2** — `client-api` 迁移或下线
4. **Phase 3** — Guest 四页 + register/forgot/reset 进 visual/aesthetic
5. **Phase 4** — `cargo run -p burncloud-loops -- css-optimize`，清零 `FAIL-*`
6. **Phase 5** — `jobs-aesthetic` 逐页（`--only-page`）；末页勾 `global_j4`；补 deploy/logs/users/connect
7. **Phase 6** — 删 09_/25_ 等无引用的 deprecated CSS

## Steve Jobs 审美 loop 用法（单页串行）

```powershell
# 仅首页（当前 pilot）
cargo run -p burncloud-loops -- jobs-aesthetic `
  --reset-page-progress `

  --max-iterations 15
```
