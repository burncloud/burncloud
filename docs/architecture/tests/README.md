## 📡 API Tests

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                                 📡 API Test Suite                                        │
├─────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                          │
│   ┌─────────────────────────────────────────────────────────────────────────────────┐    │
│   │                         Test Conventions                                         │    │
│   ├─────────────────────────────────────────────────────────────────────────────────┤    │
│   │                                                                                  │    │
│   │   ┌───────────────────────────────────────────────────────────────────────────┐  │    │
│   │   │   File Path Convention:                                                    │  │    │
│   │   │                                                                           │  │    │
│   │   │   Test file paths must correspond to route paths:                         │  │    │
│   │   │                                                                           │  │    │
│   │   │   /v1/chat/completions → tests/v1/chat/completions_test.rs               │  │    │
│   │   │   /api/users           → tests/api/users_test.rs                          │  │    │
│   │   │   /api/tokens          → tests/api/tokens_test.rs                         │  │    │
│   │   │                                                                           │  │    │
│   │   └───────────────────────────────────────────────────────────────────────────┘  │    │
│   │                                                                                  │    │
│   │   ┌───────────────────────────────────────────────────────────────────────────┐  │    │
│   │   │   Self-Bootstrapping Tests:                                                │  │    │
│   │   │                                                                           │  │    │
│   │   │   1. Start test service                                                   │  │    │
│   │   │   2. Run test cases                                                       │  │    │
│   │   │   3. Stop service                                                         │  │    │
│   │   │   4. Clean up resources                                                   │  │    │
│   │   │                                                                           │  │    │
│   │   └───────────────────────────────────────────────────────────────────────────┘  │    │
│   │                                                                                  │    │
│   │   ┌───────────────────────────────────────────────────────────────────────────┐  │    │
│   │   │   Security:                                                                │  │    │
│   │   │                                                                           │  │    │
│   │   │   No hardcoded API keys - use environment variables:                      │  │    │
│   │   │   • TEST_OPENAI_BASE_URL                                                  │  │    │
│   │   │   • TEST_OPENAI_API_KEY                                                   │  │    │
│   │   │                                                                           │  │    │
│   │   └───────────────────────────────────────────────────────────────────────────┘  │    │
│   │                                                                                  │    │
│   └─────────────────────────────────────────────────────────────────────────────────┘    │
│                                                                                          │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```

## 🚀 Running Tests

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                                 🚀 Test Commands                                         │
├─────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                          │
│   ┌─────────────────────────────────────────────────────────────────────────────────┐    │
│   │                           Cargo Test Commands                                    │    │
│   ├─────────────────────────────────────────────────────────────────────────────────┤    │
│   │                                                                                  │    │
│   │   ┌───────────────────────────────────────────────────────────────────────────┐  │    │
│   │   │   # Run all tests                                                          │  │    │
│   │   │   cargo test --all-features                                                │  │    │
│   │   │                                                                           │  │    │
│   │   │   # Run specific integration tests                                         │  │    │
│   │   │   cargo test -p burncloud-tests --test api_tests                           │  │    │
│   │   │   cargo test -p burncloud-tests e2e::                                      │  │    │
│   │   │                                                                           │  │    │
│   │   │   # Format check                                                           │  │    │
│   │   │   cargo fmt --all -- --check                                                │  │    │
│   │   │                                                                           │  │    │
│   │   │   # Lint                                                                   │  │    │
│   │   │   cargo clippy --all-targets --all-features                                │  │    │
│   │   │                                                                           │  │    │
│   │   └───────────────────────────────────────────────────────────────────────────┘  │    │
│   │                                                                                  │    │
│   └─────────────────────────────────────────────────────────────────────────────────┘    │
│                                                                                          │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```
